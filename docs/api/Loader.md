## Loader 
 
Displays a loading indicator, and on iOS an optional
loading message.
 ### Props
Name | Type | Default Value | Description
--- | --- | --- | --- 
isLoading | bool  | true | 
large | bool  | false | 
color | string  | 'gray' | 
showText | bool  | true | 
loadingText | string  | 'Loading...' | 
alignTop | bool  | false | 
inverted | bool  |   |
