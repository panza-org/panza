`Loader` (component)
====================

Displays a loading indicator, and on iOS an optional
loading message.

Props
-----

### `alignTop`

type: `bool`
defaultValue: `false`


### `color`

type: `string`
defaultValue: `'gray'`


### `inverted`

type: `bool`


### `isLoading`

type: `bool`
defaultValue: `true`


### `large`

type: `bool`
defaultValue: `false`


### `loadingText`

type: `string`
defaultValue: `'Loading...'`


### `showText`

type: `bool`
defaultValue: `true`
